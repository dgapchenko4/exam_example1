# Система управления товарами (Магазин обуви)

Веб-приложение на Django 5.2 для управления товарами с ролевой системой доступа.

## Установка и запуск

1. Активируйте виртуальное окружение:
   ```bash
   venv\Scripts\activate
   ```

2. Установите зависимости (уже установлены в venv)

3. Выполните миграции:
   ```bash
   python manage.py migrate
   ```

4. Заполните базу данных тестовыми данными:
   ```bash
   python manage.py populate_db
   ```

5. Соберите статические файлы:
   ```bash
   python manage.py collectstatic --noinput
   ```

6. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

## Тестовые пользователи

- **Администратор**: `admin` / `admin123`
- **Менеджер**: `manager` / `manager123`
- **Клиент**: `client` / `client123`

## Роли пользователей

### Гость
- Просмотр списка товаров без фильтров, сортировки, поиска

### Авторизированный клиент
- Просмотр списка товаров без фильтров, сортировки, поиска

### Менеджер
- Просмотр товаров с фильтрацией, сортировкой, поиском
- Просмотр заказов

### Администратор
- Все возможности менеджера
- CRUD операции с товарами
- CRUD операции с заказами

## Функциональность

### Товары
- **Просмотр**: Список товаров с пагинацией
- **Поиск**: По названию, описанию, категории, производителю, поставщику
- **Фильтрация**: По поставщику
- **Сортировка**: По названию, количеству (возр./убыв.)
- **CRUD**: Создание, редактирование, удаление (только администратор)

### Заказы
- **Просмотр**: Список заказов с пагинацией
- **CRUD**: Создание, редактирование, удаление (только администратор)

### Визуальные особенности
- Подсветка товаров с скидкой >15% (зеленый фон)
- Подсветка товаров не в наличии (голубой фон)
- Перечеркнутая цена с отображением скидки

## Модели данных

### Товары (Product)
- Название, категория, описание
- Производитель, поставщик
- Цена, единица измерения, количество
- Скидка, изображение

### Заказы (Order)
- Артикул, статус, пункт выдачи
- Дата заказа, дата доставки
- Клиент, позиции заказа
